# Facturacion, IVA y trazabilidad fiscal

## Objetivo

Preparar el sistema para negocios que requieren orden fiscal y seguimiento de operaciones facturables.

## Funciones

- Subtotal.
- IVA.
- Tasa de IVA.
- Precios con o sin IVA.
- Emisor fiscal.
- Clientes fiscales.
- Orden marcada para facturar.
- Seguimiento de estatus fiscal.
- Descarga/consulta de PDF/XML cuando aplica.

## Separacion operativa

La cuenta del cliente y el bloque fiscal pueden imprimirse con desglose sin afirmar que el ticket sea CFDI cuando no corresponde.

## Valor

Permite que el sistema no sea solo operativo, sino tambien comercial/fiscal, preparado para negocios formales.
