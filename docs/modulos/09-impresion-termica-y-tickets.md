# Impresion termica y tickets

## Impresoras soportadas

- Cocina.
- Barra.
- Cuenta.
- Voucher.
- Cortes.

## Transportes

- TCP RAW 9100.
- Windows spooler.
- Simulacion para pruebas.

## Tickets configurables

El sistema permite configurar formatos de:

- Ticket de cuenta.
- Cocina.
- Barra.
- Corte de mesero.
- Corte final.
- Corte de propinas.
- Voucher.

## Cajon de dinero

La apertura del cajon se integra al flujo de cobro, pero solo cuando corresponde:

- Efectivo.
- Mixto con efectivo.

## App movil

La app movil puede imprimir en impresoras de red usando configuracion local por dispositivo.

## Valor tecnico

La impresion es critica en restaurantes. El sistema separa estaciones, formatos y transportes para adaptarse a escenarios reales de hardware.
