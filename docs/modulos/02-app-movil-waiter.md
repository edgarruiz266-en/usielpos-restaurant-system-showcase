# App movil Flutter

## Proposito

La app movil no es solo una comanda. Tambien funciona como herramienta de supervision, administracion y cobro segun permisos.

## Roles

- Mesero: mesas, pedidos, envio a cocina/barra, cobro segun permisos.
- Supervisor: supervision, cortes, autorizaciones, inventario y movimientos.
- Administrador: configuracion, empleados, productos, paquetes, salas, mesas e inventario.

## Modulos implementados

- Login/licencia.
- Mesas y salas con refresco en vivo.
- Pedidos.
- Revision de orden.
- Cancelaciones parciales/totales.
- Cobro.
- Mercado Pago Point.
- Impresion local ESC/POS TCP.
- Cortes.
- Propinas.
- Productos.
- Paquetes.
- Empleados.
- Salas y mesas.
- Inventario, recetas BOM y kardex.
- Configuracion local de impresoras.
- Edicion local de tickets.

## Conexion

La app puede operar contra:

- API remota en la nube.
- API local dentro de la red del cliente.

La fuente activa se guarda por dispositivo y debe ser mutuamente excluyente: LOCAL o NUBE.

## Valor comercial

La app permite que el dueno o supervisor no dependa de estar fisicamente en la sucursal para revisar cortes, descuentos, cancelaciones, inventario, empleados, productos y movimientos relevantes.
