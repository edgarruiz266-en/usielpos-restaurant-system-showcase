# Administracion, configuracion y operacion

## Administracion desde PC y movil

Segun permisos, el sistema permite administrar:

- Empleados.
- Productos.
- Categorias.
- Paquetes.
- Componentes.
- Precios.
- Salas.
- Mesas.
- Inventario.
- Recetas.
- Tickets.
- Impresoras.
- Configuracion fiscal.
- Configuracion Mercado Pago Point.

## Filosofia

El sistema busca que la operacion no quede encerrada en una sola PC. La app movil tambien puede servir para administrar y supervisar cuando el rol lo permite.

## Owner/Admin Panel

La arquitectura esta preparada para crecer hacia paneles de administracion mas amplios, control de clientes, licencias, configuraciones y auditoria global.
