# Seguridad, roles y permisos

## Roles principales

- Administrador.
- Supervisor.
- Mesero.

## Controles

- Login.
- PIN para acciones sensibles.
- Autorizacion en descuentos.
- Autorizacion en cancelaciones.
- Permisos por modulo.
- Auditoria de acciones.

## Acciones sensibles

- Cancelar productos enviados.
- Cancelar orden.
- Aplicar descuentos personalizados.
- Autorizar tarjeta manual cuando Point esta activo.
- Ajustar inventario.
- Modificar configuraciones.

## API

La API usa autorizacion por token y licencia para validar contexto de cliente y usuario.

## Valor

El sistema no solo controla acceso; tambien deja evidencia de operaciones delicadas.
