## Proyecto destacado

### UsielPOS Restaurant System

Sistema integral para restaurantes con:

- POS Desktop en Python/PyQt5.
- App movil Flutter para meseros, supervisores y administradores.
- API REST con FastAPI.
- Azure SQL / SQL Server.
- Mercado Pago Point.
- Inventarios con recetas/BOM.
- Paquetes comerciales.
- Facturacion CFDI.
- Impresion termica.
- Cortes, propinas y auditoria.

[Ver caso de estudio](https://github.com/TU_USUARIO/usielpos-restaurant-system-showcase)
