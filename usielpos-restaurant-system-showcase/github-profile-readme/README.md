## Proyecto destacado

### UsielPOS Restaurant System

Sistema integral para restaurantes en operación real.

- POS Desktop con Python/PyQt5.
- App móvil Flutter para meseros.
- API REST con FastAPI.
- Azure SQL como base de datos.
- Integración con Mercado Pago Point.
- Inventarios, paquetes, cortes, impresión térmica y CFDI.

[Ver caso de estudio](https://github.com/edgarusiel/usielpos-restaurant-system-showcase)
