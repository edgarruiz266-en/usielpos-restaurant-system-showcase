# Impresión térmica

El sistema integra impresión térmica para tickets internos y documentos operativos.

## Tickets soportados

- Cocina.
- Barra.
- Cuenta del cliente.
- Voucher de mesero.
- Corte de caja.
- Corte por mesero.
- Corte de propinas.

## Tecnologías y protocolos

- ESC/POS.
- Impresión por red TCP 9100.
- Impresión por spooler de Windows.
- Configuración local por dispositivo.

## Diseño tolerante a fallos

La impresión no debe romper operaciones críticas. Si una impresora falla, el sistema debe informar claramente y permitir continuar cuando la regla de negocio lo permita.
