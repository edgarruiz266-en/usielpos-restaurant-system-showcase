# Facturación CFDI

El sistema incluye preparación fiscal para operación en México, separando subtotal, IVA y total, además de datos fiscales del emisor y cliente.

## Funciones principales

- Configuración de emisor fiscal.
- Registro de clientes fiscales.
- Cálculo de subtotal, IVA y total.
- Marcado de órdenes que requieren factura.
- Preparación para emisión y descarga de PDF/XML.
- Leyendas fiscales en tickets cuando aplica.

## Consideraciones

- No subir XML/PDF reales a este repositorio.
- No mostrar RFCs reales ni UUIDs fiscales.
- Usar datos de prueba para capturas.
