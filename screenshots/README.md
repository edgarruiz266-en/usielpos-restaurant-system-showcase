- Tokens.
- Licencias reales.
- Correos o telefonos de clientes.
- RFC reales.
- UUID fiscales reales.
- Datos bancarios.
- IPs privadas si no quieres mostrarlas.
- Nombres de negocios reales si no tienes permiso.
