# Checklist antes de publicar

Antes de subir capturas, videos o documentos a GitHub, revisar:

- [ ] No hay tokens de Mercado Pago.
- [ ] No hay cadenas de conexion.
- [ ] No hay claves de Azure.
- [ ] No hay RFC reales.
- [ ] No hay datos de clientes reales.
- [ ] No hay correos ni telefonos reales.
- [ ] No hay licencias reales.
- [ ] No hay nombres de empleados reales sin permiso.
- [ ] No hay tickets con datos fiscales reales.
- [ ] No hay URLs privadas de administracion.
- [ ] No hay IPs internas sensibles.
